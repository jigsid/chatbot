How to Integrate

To add this chatbot to your website, simply add the following script tag to your HTML:

<script src="https://your-domain.com/embed.js"></script>

You can also pre-set the user's email if they're already logged in:

<script>
  window.smartRepUserEmail = "user@example.com";
</script>
<script src="https://your-domain.com/embed.js"></script>

JavaScript API

The chatbot exposes a JavaScript API that you can use to control it:

// Open the chatbot
SmartRepChat.open();

// Close the chatbot
SmartRepChat.close();

// Set the user's email
SmartRepChat.setUserEmail("user@example.com");
